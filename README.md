# AI-Powered Creative Studio

An AI-based creative platform that allows users to generate images, apply style transfer, and create AI-generated videos using modern web technologies and AI models.

This project is designed to showcase real-world usage of AI in creative workflows, combining frontend, backend, and AI integration into a single platform.

---

## Features

-  **AI Image Generation**
  - Generate high-quality images from text prompts
  - Supports different styles and creative variations

-  **Style Transfer**
  - Apply artistic styles to images using AI
  - Transform normal images into artwork-style visuals

-  **AI Video Generation**
  - Generate short videos using AI-based rendering
  - Combines frames and effects programmatically

-  **Modern UI**
  - Clean and responsive user interface
  - Optimized for desktop and mobile devices

-  **Scalable Architecture**
  - Separate frontend and backend
  - API-based communication
